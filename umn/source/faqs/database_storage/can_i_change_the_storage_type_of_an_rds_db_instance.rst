Can I Change the Storage Type of an RDS DB Instance?
====================================================

Yes. After an RDS DB instance is created, you can change its storage type.

For RDS for MySQL instances, see :ref:`Changing a Storage Type <rds_05_0140>`.

For RDS for PostgreSQL instances, see :ref:`Changing a Storage Type <rds_pg_05_0084>`.

For RDS for SQL Server instances, see :ref:`Changing a Storage Type <rds_sqlserver_05_0050>`.
